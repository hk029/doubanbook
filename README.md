# doubanbook
---
## 介绍

doubanbook是一个爬虫，它实现的功能是爬取豆瓣读书9分榜单

这是一个scrapy实例，用来熟悉掌握scrapy的功能和用法。

## 1.0功能
爬取豆瓣读书9分榜单

## 用法
在setting.py中修改csv保存的路径，然后运行python main.py
